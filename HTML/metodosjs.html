Métodos comunes de JavaScript

Tipo / Objeto Método Descripción
String toUpperCase() Convierte la cadena a mayúsculas
String toLowerCase() Convierte la cadena a minúsculas
String slice() Extrae una sección de la cadena
String replace() Reemplaza parte de la cadena
Array push() Añade uno o más elementos al final del array
Array pop() Elimina el último elemento del array
Array shift() Elimina el primer elemento del array
Array unshift() Añade uno o más elementos al inicio del array
Array map() Crea un nuevo array con los resultados de una función aplicada a cada elemento
Array filter() Crea un nuevo array con los elementos que cumplen una condición
Object keys() Devuelve un array con las claves del objeto
Object values() Devuelve un array con los valores del objeto
Object entries() Devuelve un array con pares [clave, valor]
Math random() Devuelve un número aleatorio entre 0 y 1
Math floor() Redondea hacia abajo
Math ceil() Redondea hacia arriba
Math round() Redondea al entero más cercano
