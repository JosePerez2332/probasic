¿Qué son las etiquetas semánticas?

Las etiquetas semánticas en HTML sirven para definir una estructura clara de mi sitio web.

Tabla de etiquetas semánticas

ETIQUETA DESCRIPCIÓN
<header> Define una sección de contenido introductorio o de navegación, a menudo ubicada en la parte superior de una página web o de una sección, es una parte visible del cuerpo de la página y es lo primero que ven los usuarios al visitarla.
<nav> Se usa para agrupar enlaces de navegación principales en un sitio web, indicando a navegadores y lectores de pantalla que el contenido es para la navegación del sitio, mejorando la accesibilidad, el SEO y la legibilidad del código.
<main> se utiliza para encapsular el contenido principal y único de un documento web.
<footer> Se usa para definir el pie de página de un sitio web o de una sección específica del documento. Contiene información que complementa el contenido principal, como derechos de autor, enlaces a páginas legales (políticas de privacidad, términos de uso), información de contacto, el logo de la empresa o formularios de suscripción.
<section> Se usa para agrupar contenido temáticamente relacionado dentro de un documento, creando secciones coherentes que mejoran la estructura, la legibilidad, la accesibilidad y el SEO.
<aside> Define contenido relacionado indirectamente con el contenido principal de la página, como barras laterales, anuncios, citas destacadas, biografías de autores, o enlaces relacionados. Sirve para agrupar este contenido adicional y separarlo del contenido central, mejorando la organización semántica de la página web y la experiencia del usuario al mantener el foco en la información importante.
<article> Se usa para encerrar un contenido que es independiente y autónomo, es decir, tiene sentido y puede ser reutilizado o distribido por sí solo fuera del contexto del resto de la página. Se aplica a elementos como artículos de blog, noticias, publicaciones de foros o comentarios de usuarios. Aporta un valor semántico que ayuda a los motores de búsqueda y a los desarrolladores a entender la estructura y la importancia de la información en la web.
